# 심야 병동 전략 난이도·타임어택 설계

- 문서 버전: 0.3
- 기준일: 2026-07-24
- 상태: 서버 규칙 1차 구현 및 밸런스 검증 진행 중
- 관련 현재 구현: `docs/spec/balance.md`, `src/shared/progression.ts`, `src/server/engine.ts`

이 문서는 귀신의 HP와 공격력만 높이는 수치형 난이도에서 벗어나, 난이도마다 요구하는 공략을 다르게 만드는 규칙형 난이도를 정의한다. 타임어택·제어 적응·방어막·방향성 보호막 상태는 현재 서버 스냅샷에서 관리하며, 세부 수치는 플레이 테스트와 서버 시뮬레이션으로 조정한다.

## 1. 해결하려는 문제

현재는 다음 조합이 거의 모든 난이도에서 같은 방식으로 작동한다.

```text
다수 서리 스프레이 감속
→ HP 20% 이하 봉쇄 그물 속박
→ 다수 수호 포탑 집중 사격
```

귀신의 HP만 높이면 전투 시간이 늘어나고, 플레이어는 그동안 골드와 전력을 더 생산해 같은 조합을 더 많이 설치할 수 있다. 반복 제어에만 저항을 주는 방식도 첫 접근에서 최대 포탑 화력으로 처치하는 문제를 해결하지 못한다.

따라서 고난도는 아래 네 축을 함께 사용한다.

1. 기본 HP·공격력·이동속도
2. 제어 적응
3. 한 번의 폭딜로 처치할 수 없는 방어 단계
4. 맵·장비·협동을 요구하는 스테이지 규칙

## 2. 설계 원칙

1. 쉬움과 노말은 현재의 자유로운 건설 공식을 학습하는 구간으로 유지한다.
2. 타임어택은 악몽부터만 등장한다. 쉬움과 노말에서는 절대 등장하지 않는다.
3. 완전 면역은 사전 안내가 있는 고유 변종을 제외하고 사용하지 않는다. 기본 대응은 점감, 방어막, 방향성 보호막으로 제공한다.
4. 봉쇄 그물은 여전히 HP 20% 이하에서 1.5초 속박을 시작하지만, 제어 적응 수치에 따라 실제 지속시간이 감소할 수 있다.
5. 리스폰 왕복 시간을 기준으로 한 짧은 시간제 그물 면역은 사용하지 않는다. 대신 한 번의 문 공격 시도에서 그물은 한 번만 발동한다.
6. 황금 포탑이나 전투 보급품이 요구되는 스테이지는 랜덤 획득 또는 유료 구매만으로 진입 조건을 충족하게 만들지 않는다. 도전 전에 확정 획득 경로와 권장 장비를 안내한다.
7. 같은 난이도 안에서도 단일 공략이 모든 스테이지를 해결하지 못하도록 규칙 조합을 순환한다.

## 3. 난이도 티어별 전략 목표

| 티어 | 타임어택 확률 | 기본 특수 규칙 | 요구 전략 |
| --- | ---: | --- | --- |
| 쉬움 | 0% | 없음 | 이동, 점유, 건설 학습 |
| 노말 | 0% | 없음 | 문·침대·포탑 강화 학습 |
| 악몽 | 7% | 제어 적응 | 감속 설비 수와 폭딜 타이밍 조절 |
| 지옥 | 12% | 제어 적응 + 방어막 1개 | 최소 두 번의 유효 공격 기회 확보 |
| 불지옥 | 18% | 방향성 보호막 또는 방어막 2개 | 인접 방 교차 사격 또는 보호막 관통 |
| 에픽 | 25% | 방어막 3개 + 자원 압박 | 전투 보급품과 역할 분담 |
| 신화 | 30% | 방어막 4개 + 규칙 2개 조합 | 황금 포탑·협동·보급품 중 하나 이상 |
| 레전더리 | 35% | 규칙 2~3개 + 선택적 타임어택 | 최고 성장과 숙련된 팀 조합 |

타임어택 확률은 일반 진행 모드 기준이다. 랭크전에서는 무작위 확률을 사용하지 않고 48시간 시즌 계약에 적용 여부를 미리 고정한다.

## 4. 제어 적응 게이지

### 4.1 상태

고난도 귀신은 `controlResolve` 값을 `0~100` 범위로 가진다.

- 리스폰 구역으로 후퇴하거나 회복해도 초기화하지 않는다.
- 한 매치 안에서는 자연 감소하지 않는다.
- 100에 도달해 제어를 해제한 뒤에는 50으로 돌아간다.
- 게이지는 플레이어에게 귀신 HP바 아래 작은 보조 게이지로 표시한다.

### 4.2 서리 스프레이 증가량

귀신이 서리 스프레이 범위 안에 있는 동안 실제 경과시간을 기준으로 증가한다.

| 동시에 적용 중인 서리 스프레이 | 초당 증가량 |
| ---: | ---: |
| 1개 | 12 |
| 2개 | 30 |
| 3개 이상 | 54 |

다수 설치는 더 강한 순간 감속을 제공하지만 제어 해제를 빠르게 유발한다.

### 4.3 봉쇄 그물 판정

봉쇄 그물이 발동하면 `controlResolve`에 60을 더한 값을 기준으로 실제 속박 시간을 결정한다.

| 그물 적용 후 게이지 | 실제 속박 시간 |
| ---: | ---: |
| 0~69 | 1.5초 |
| 70~99 | 0.9초 |
| 100 | 0.45초 |

게이지가 100에 도달하면 해당 속박이 끝난 직후 모든 감속을 해제하고 2.5초 동안 감속·속박에 면역이 된다. 이후 게이지는 50으로 돌아간다.

한 방의 봉쇄 그물은 귀신이 그 방의 문을 목표로 선택한 한 번의 공격 시도에서 한 번만 발동한다. 귀신이 리스폰 회복을 마치고 새 목표를 선택하면 새로운 공격 시도로 간주한다.

## 5. 방어막 단계

제어 적응만으로는 첫 접근에서 발생하는 최대 포탑 폭딜을 막을 수 없으므로 지옥 이상 일부 스테이지에 소모형 방어막을 부여한다.

### 5.1 기본 규칙

- 귀신은 난이도에 따라 `barrierLayers`를 가진다.
- 귀신 HP가 0이 되었을 때 방어막이 남아 있으면 사망하지 않는다.
- 방어막 한 겹을 소비하고 HP를 1로 고정한다.
- 모든 속박과 감속을 즉시 해제하고 0.8초 동안 안개 상태가 된다.
- 안개 상태에서는 공격 대상이 되지 않으며 가장 가까운 리스폰 구역으로 즉시 후퇴한다.
- 파괴된 방어막은 회복 구역에서 복구되지 않는다.
- 방어막이 모두 파괴된 뒤 HP가 0이 되면 정상적으로 처치된다.

이 규칙은 한 번의 최대 포탑 집중 사격으로 여러 방어막을 동시에 제거할 수 없도록 한다.

### 5.2 화면 표시

- HP바 옆에 남은 방어막 수만큼 작은 방패 아이콘을 표시한다.
- 방어막 파괴 시 화면 흔들림보다 깨지는 방패 효과와 명확한 효과음을 우선한다.
- 안개 후퇴가 무적 버그로 보이지 않도록 `방어막 파괴` 전투 메시지를 표시한다.

### 5.3 관통 수단

- 황금 심판 포탑은 방어막에 추가 피해를 주거나 고유 재사용시간마다 방어막을 하나 추가 파괴할 수 있다.
- 차기 전투 보급품 `보호막 교란기` 또는 `EMP`는 남은 방어막을 하나 제거한다.
- 관통 수단의 구체적인 비용과 획득 경로는 `docs/spec/shop-consumables.md`에서 정의한다.

## 6. 방향성 보호막과 협동 스테이지

두 플레이어가 가까운 방을 점유해야 유리한 스테이지에서는 방향성 보호막을 사용한다.

### 6.1 피해 판정

- 귀신은 현재 공격 중인 문 방향에 보호막을 전개한다.
- 공격 중인 방 안의 포탑 피해는 35%만 적용한다.
- 다른 점유 방에서 들어오는 포탑 피해는 100% 적용한다.
- 서로 다른 두 방의 포탑이 3초 동안 동시에 명중하면 방향성 보호막을 6초 동안 해제한다.
- 황금 심판 포탑은 방향성 보호막의 피해 감소를 50% 무시한다.
- 보호막 교란 보급품은 일정 시간 방향성 보호막을 강제로 해제한다.

### 6.2 맵 조건

- 해당 규칙이 적용되는 스테이지에는 서로의 문 앞 복도를 사거리 안에 둘 수 있는 인접 방 쌍을 최소 하나 생성한다.
- 친구랑하기에서는 두 플레이어가 인접 방을 선택할 수 있도록 대기실 또는 게임 시작 안내에 추천 방을 표시한다.
- 혼자하기에서는 같은 규칙을 AI 동료가 수행할 수 있거나 황금 포탑·보급품으로 대체 공략할 수 있어야 한다.

### 6.3 쌍둥이 변종 확장안

- 쌍둥이는 서로 다른 문을 동시에 공격한다.
- 한쪽이 처치된 뒤 다른 쪽을 10초 안에 처치하지 못하면 먼저 처치된 쌍둥이가 일부 HP로 부활한다.
- 이 규칙은 별도 스테이지 수정자로 표시하며 일반 쌍둥이 변종에는 자동 적용하지 않는다.

## 7. 공략 경로

고난도 스테이지는 다음 중 두 개 이상의 공략 경로를 제공한다.

| 공략 경로 | 역할 |
| --- | --- |
| 일반 포탑 성장 | 충분한 경제와 반복 공격으로 방어막을 순차 제거 |
| 황금 심판 포탑 | 방어막·방향성 보호막을 빠르게 관통 |
| 전투 보급품 | EMP, 과충전, 긴급 수리 등으로 특정 약점 보완 |
| 인접 방 협동 | 교차 사격으로 방향성 보호막 해제 |
| 자원 특화 빌드 | 초반 위험을 감수하고 후반 성장 속도를 확보 |

한 스테이지가 랜덤 상자에서 황금 티켓이 나오지 않으면 절대 클리어할 수 없는 구조가 되어서는 안 된다.

### 7.1 황금 심판 포탑 획득 경계

- 혼자하기·친구랑하기에서는 베테랑 등급과 지정 고난도 도전 과제를 통한 영구 설계도 해금 경로를 제공한다.
- 황금 티켓은 영구 설계도를 대체하는 유일한 경로가 아니라 행운 보상 또는 추가 설치권으로 사용한다.
- 정확한 영구 해금 스테이지와 도전 과제는 방어막·방향성 보호막 전투 로그를 수집한 뒤 확정한다.
- 황금 포탑은 경기당 기본 1대로 제한하고 방어막 관통에 특화한다. 일반 포탑보다 항상 효율적인 단순 상위 호환으로 만들지 않는다.
- 랭크전에서는 개인의 해금 여부를 무시하고 계약별 `사용 불가 / 동일 대여 / 현장 해금 / 점수 페널티` 정책을 사용한다.
- 세부 시즌 규칙은 `docs/spec/ranked-mode.md`를 따른다.

## 8. 타임어택 이벤트

### 8.1 상태 흐름

```text
LOBBY
→ EVENT_INTRO 2초
→ COUNTDOWN 30초
→ PLAYING 5분
→ OVERTIME
→ VICTORY 또는 DEFEAT
```

타임어택에 당첨되지 않은 매치는 기존처럼 `LOBBY → COUNTDOWN → PLAYING`으로 진행한다.

### 8.2 이벤트 인트로

`EVENT_INTRO` 동안 이동, 봇 AI, 귀신 AI, 자원 생산, 건설 입력을 모두 정지한다.

- 중앙 큰 빨간 글씨: `Time Attack`
- 안내 문구:

```text
당신에게 5분의 시간이 주어졌습니다.
5분안에 귀신을 물리치고 탈출하세요.
```

- 2초 후 전체 안내가 서서히 사라지고 30초 준비 타이머가 시작된다.
- 30초 준비 타이머가 끝나면 좌측 하단에 검은 배경과 빨간 글씨로 `05:00`을 표시한다.

### 8.3 제한시간과 오버타임

- 5분 제한시간은 `PLAYING` 전환 시점부터 서버 시간으로 계산한다.
- 0초가 되어도 즉시 패배하지 않고 `OVERTIME`에 진입한다.
- 오버타임 시작 시 귀신 시각 크기를 한 번만 2배로 만든다.
- 오버타임 시작과 이후 1분마다 최대 HP, 현재 HP, 공격력을 각각 2배로 만든다.
- 공격속도도 2배로 올리되 실제 공격 간격은 최소 0.20초로 제한한다.
- HP 배율 적용 시 현재 HP 비율을 보존한다.
- 시각 크기는 오버타임 스택마다 반복해서 커지지 않는다.
- 타이머는 0 이후 `+00:01`, `+01:00` 형식으로 초과 시간을 표시한다.

### 8.4 보상과 공정성

- 타임어택 클리어는 기본 포인트와 XP에 35% 보너스를 지급한다.
- 일반 진행에서 이벤트 발생 여부는 매치 시작 시 서버가 결정하고 스냅샷에 저장한다.
- 재접속으로 이벤트 결과가 다시 추첨되지 않는다.
- 랭크전에서는 이벤트를 무작위로 추첨하지 않고 같은 계약의 모든 참가자에게 동일하게 적용한다.

## 9. 서버 데이터 제안

```ts
type MatchModifier = "none" | "time-attack";

interface DifficultyRuleState {
  modifier: MatchModifier;
  introRemaining: number;
  timeAttackRemaining: number | null;
  overtimeStacks: number;
}

interface GhostControlState {
  controlResolve: number;
  controlImmuneUntil: number;
  barrierLayers: number;
  directionalShieldActive: boolean;
}
```

실제 이름은 현재 스냅샷 구조에 맞춰 변경할 수 있지만, 이벤트 시간과 귀신 제어 상태는 반드시 서버 권위 상태로 관리한다.

## 10. 밸런스 목표와 검증

목표 클리어율은 충분한 표본을 모은 뒤 다음 범위로 조정한다.

| 티어 | 목표 클리어율 |
| --- | ---: |
| 쉬움 | 85~95% |
| 노말 | 75~90% |
| 악몽 | 50~65% |
| 지옥 | 35~50% |
| 불지옥 | 25~40% |
| 에픽 | 15~30% |
| 신화 | 8~20% |
| 레전더리 | 3~15% |

서버 로그에는 최소한 다음 항목을 남긴다.

- 스테이지와 활성 규칙
- 클리어 여부와 전투 시간
- 설치·강화한 건물 수
- 귀신의 감속 유지율과 실제 속박 시간
- 파괴한 방어막 수
- 방별 포탑 피해와 교차 사격 시간
- 사용한 전투 보급품
- 플레이어 사망 시점과 문 잔여 HP

자동 시뮬레이션은 `서리 스프레이 + 봉쇄 그물 + 일반 포탑 최대 설치` 조합이 모든 고난도 규칙에서 높은 승률을 내지 않는지 검증한다. 특정 조합의 전 티어 평균 승률이 70%를 넘으면 규칙 분산이 실패한 것으로 간주한다.

# 심야 병동 게임 모드·랭크전 설계

- 문서 버전: 0.4
- 기준일: 2026-07-24
- 상태: S1 알파 구현 진행 중
- 관련 현재 구현: `docs/spec/map-multiplayer.md`, `src/server/auth.ts`, `src/server/RankedQueue.ts`, `src/server/GameRoom.ts`, `src/server/engine.ts`

이 문서는 기존 싱글·멀티플레이의 사용자 표시 명칭을 변경하고, 일반 성장 등급과 분리된 14일 시즌형 랭크전을 정의한다. 일반 협동의 편안한 보상 구조는 유지하되 랭크전은 모든 참가자가 같은 조건에서 계약 점수를 겨루도록 설계한다.

## 1. 플레이 방식과 등급 표시

| 내부 모드 | 사용자 표시 | 등급 기준 | 역할 |
| --- | --- | --- | --- |
| `solo` | 혼자하기 | 혼자하기 진행 등급 | 봇과 함께 진행하는 일반 PvE |
| `multiplayer` | 친구랑하기 | 친구랑하기 진행 등급 | 초대 코드로 참가하는 비공개 협동 PvE |
| `ranked` | 랭크전 | 현재 시즌 랭크 | 4인 대기열에서 동일 계약 점수를 겨루는 경쟁 협동 PvE |

사용자 화면에서는 `혼자하기`, `친구랑하기`, `랭크전`만 사용한다. 선택값은 `accounts.selected_play_mode`에 저장해 다음 접속에도 복원한다.

### 1.1 홈 프로필 카드

홈 좌측 상단의 인게임 라벨을 누르면 별도의 표시 설정을 연다. 게임 시작에 사용할 플레이 방식과 인게임에 보일 뱃지는 서로 독립적으로 저장한다.

| 선택 상태 | 표시 예시 | 배지 |
| --- | --- | --- |
| 혼자하기 | `혼자하기 · 고수` | 혼자하기 진행 등급 배지 |
| 친구랑하기 | `친구랑하기 · 하수` | 친구랑하기 진행 등급 배지 |
| 랭크전 배치 전 | `랭크전 · 배치 전` | 배치 전 시즌 배지 |
| 랭크전 배치 완료 | `랭크전 · 골드 II` | 현재 시즌 랭크 배지 |

- 혼자하기·친구랑하기 중 높은 등급을 홈에 고정 표시하지 않는다.
- 현재 선택한 플레이 방식은 `accounts.selected_play_mode`에, 인게임 라벨의 표시 등급은 `accounts.profile_display_mode`에 각각 저장하고 다음 접속에도 복원한다.
- 현재 스테이지 카드는 등급 카드와 분리한다.
- 대기실과 인게임 이름표는 `profile_display_mode`로 지정한 등급과 배지를 사용한다. 전투 보정·스테이지 진행·매칭은 실제 입장한 플레이 방식의 데이터만 사용한다.
- 왕관 모양 랭킹 진입점은 일반 진행 XP가 아니라 현재 랭크 시즌 순위표를 연다.
- 칭호 슬롯은 같은 표시 설정 패널에서 관리한다. 현재 보유 칭호가 없으면 `칭호 없음`으로 표시하며, 시즌 보상·업적 칭호 보상 구현 시 이 슬롯에만 선택지를 추가한다.

## 2. 일반 협동 결과와 랭크 결과 분리

일반 매치는 모든 귀신을 처치하면 팀 전체가 승리한다. 먼저 사망한 플레이어도 남은 팀원이 클리어하면 같은 일반 진행 보상을 받는다. 이 규칙은 혼자하기·친구랑하기에는 유지하지만 개인 실력을 비교하는 랭크 점수에는 사용하지 않는다.

다음 데이터는 서로 직접 변환하지 않는다.

- 혼자하기 XP·등급·스테이지 진행
- 친구랑하기 XP·등급·스테이지 진행
- 커스텀 포인트와 일반 전투 보급품 재고
- 랭크 시즌 점수·티어·순위
- 향후 추가할 2인 고정 팀 랭크 기록

## 3. 랭크전 참가 조건

랭크전 참가 자격은 장비 보유량이 아니라 기본 숙련도로 판정한다.

1. 혼자하기 `노말 5` 클리어
2. 일반 게임 유효 완료 10회 이상
3. 랭크전 규칙 튜토리얼 완료
4. 중도 이탈 또는 운영 제재 상태가 아님
5. 시즌 배치 계약 5회 완료 후 정식 티어와 순위 부여

황금 심판 포탑, 특정 캐릭터, 스킨 또는 유료 상품 보유는 참가 조건으로 사용하지 않는다. 현재 서버는 `노말 5`를 실제로 클리어해 다음 스테이지가 해금된 상태와, 랭크 결과를 제외한 일반 매치 완료 10회를 모두 확인한다.

### 3.1 랭크 대기열과 자동 시작

- 랭크전은 `혼자하기` 방을 만들지 않는다. 현재 시즌·계약별 대기열에 입장해 같은 계약의 참가자와 매칭한다.
- 기본 탐색 범위는 내 RP 기준 `±150`이며, 10초마다 `±50`씩 넓혀 최대 `±500 RP`까지만 허용한다.
- 실제 플레이어 4명이 같은 범위에 모이면 봇 없이 즉시 같은 방으로 배정한다.
- 입장 후 40초 안에 4명이 채워지지 않으면, 당시 대기열의 실제 플레이어를 유지하고 빈 자리만 봇으로 보충해 시작한다.
- 매칭 완료 뒤에는 준비·봇 추가·방장 시작 버튼을 제공하지 않는다. 배정된 실제 플레이어가 모두 연결되는 순간 서버가 카운트다운을 시작한다.
- 대기열 화면은 경과 시간, `n/4` 참가 수, 현재 탐색 RP 범위와 40초 보충 규칙을 표시한다. 취소는 매칭 완료 전까지만 가능하다.

## 4. 14일 시즌과 계약 구성

### 4.1 시즌 일정

- 한 시즌은 한국 시간 기준 14일 동안 진행한다.
- 시즌 ID는 `S1`, `S2`, `S3` 순서로 증가한다.
- 시즌은 월요일 00:00에 시작해 2주 뒤 일요일 23:59:59에 종료한다.
- 표시 랭크, 시즌 점수와 현재 순위표는 시즌마다 초기화한다.
- 지난 시즌 최종 순위와 수상자는 명예의 전당에 영구 보존한다.
- 시즌 종료 뒤 기록 검증 시간 동안 순위표를 동결하고 보상 지급을 보류한다.

초기화되는 것은 경쟁용 표시 점수와 순위표다. 일반 진행 등급, 포인트, 보유 캐릭터·스킨·보급품과 과거 시즌 칭호는 초기화하지 않는다.

### 4.2 시즌 계약

- 48시간마다 새로운 랭크 계약을 하나 공개해 한 시즌에 총 7개를 제공한다.
- 각 계약은 모든 참가자에게 같은 맵 시드, 귀신 변종, 스테이지, 난이도 규칙과 타임어택 적용 여부를 제공한다.
- 계약은 제한시간 안에 횟수 제한 없이 재도전할 수 있다.
- 각 계약에서 달성한 최고 점수만 저장한다.
- 시즌 점수는 7개 계약 중 가장 높은 5개 계약 점수의 합이다.
- 두 계약을 놓쳐도 상위권 도전이 가능해 접속 일수보다 실력과 개선 정도를 우선한다.
- S1은 4인 자동 매칭 협동 계약을 운영하되, 시즌 순위와 보상은 개인 계약 점수로 계산한다.
- 향후 2인 고정 팀 랭크를 추가할 때는 현재 개인 순위와 별도 시즌 점수·순위표·보상을 사용한다.

### 4.3 고정 전투 조건

- 랭크전의 캐릭터·스킨 전투 특성은 계약별 표준 능력치로 보정한다.
- 시즌 한정 스킨의 고유 능력은 혼자하기·친구랑하기에서만 적용하고 랭크전에서는 외형만 적용한다.
- 전투 보급품은 계약이 같은 종류와 수량을 대여하거나, 개인 재고 사용량만큼 점수 페널티를 적용한다.
- 무작위 타임어택을 사용하지 않는다. 타임어택 계약이면 입장 전부터 명확히 표시한다.
- 서버가 계약 시드, 허용 장비와 최종 점수를 권위적으로 관리한다.

### 4.4 현재 서버 구현 범위

- `S1`은 2026-07-20 월요일 00:00 KST를 기준으로 14일 단위로 증가한다. 계정의 시즌 ID가 바뀌면 RP·배치 수·계약 수를 800 RP·0회로 초기화한다.
- 계약 번호는 시즌 시작 기준 48시간 창으로 계산한다. 같은 `Sx-Cy` 계약은 FNV 기반 고정 시드로 같은 맵을 생성한다.
- S1 알파는 계약별 Durable Object 대기열에서 최대 4명의 실제 참가자를 매칭한다. 계약마다 악몽 스테이지·타임어택 여부·황금 포탑 정책·보급품 정책을 서버에서 고정한다.
- 각 계약 대기열은 40초 뒤에만 부족한 인원을 봇으로 보충한다. 실제 참가자 4명이 찬 매치에는 봇을 절대 추가하지 않는다.
- 시즌 종료 순위 동결, 우편 보상 지급, 명예의 전당과 운영 제재 심사는 다음 서버 마일스톤이다. 아래 보상 표는 이 자동 지급 기능이 들어갈 때의 확정 정책이다.

## 5. 황금 심판 포탑 정책

### 5.1 랭크전

개인이 일반 모드에서 황금 심판 포탑을 해금했는지는 랭크 성능에 영향을 주지 않는다. 계약은 다음 정책 중 하나를 명시한다.

| 정책 | 규칙 |
| --- | --- |
| 사용 불가 | 모든 참가자의 황금 포탑 설치를 막는다. |
| 계약 대여 | 모든 참가자에게 동일한 황금 포탑 설계도 1개를 지급한다. |
| 현장 해금 | 계약의 동일한 전투 목표를 달성하면 경기 중 설계도를 해금한다. |
| 제한 사용 | 모두가 사용할 수 있지만 설치 또는 강화 시 계약 점수를 차감한다. |

- 개인 황금 티켓과 영구 설계도는 랭크 계약으로 가져올 수 없다.
- 황금 포탑이 공략상 필요한 계약은 모든 참가자에게 확정적인 대여·현장 해금 경로를 제공한다.
- 랜덤 상자나 유료 구매 결과로 랭크 계약의 클리어 가능 여부가 갈리지 않게 한다.

### 5.2 혼자하기·친구랑하기

황금 심판 포탑은 고난도 진행 보상으로 유지하되 무작위 황금 티켓만이 유일한 획득 경로가 되어서는 안 된다.

- 베테랑 등급과 지정 고난도 도전 과제를 포함하는 영구 설계도 해금 경로를 제공한다.
- 정확한 스테이지와 도전 과제는 고난도 전투 로그를 수집한 뒤 확정한다.
- 황금 티켓은 영구 설계도 이전의 행운 보상 또는 추가 설치권으로 유지할 수 있다.
- 기본 설치 한도는 경기당 1대로 제한한다.
- 일반 포탑의 단순 상위 호환이 아니라 방어막·방향성 보호막 관통에 특화한다.
- 높은 전력 유지비, 큰 설치 부하 또는 느린 사격 같은 명확한 기회비용을 부여한다.

## 6. 계약 점수와 순위

클리어 성공을 가장 크게 평가하고, 같은 계약 안에서 남은 시간과 방어 상태로 세부 순위를 정한다.

```text
계약 점수
= 스테이지 기본 점수
+ 남은 시간 보너스
+ 평균 문 잔여 HP 보너스
+ 생존 보너스
+ 계약 특별 목표 보너스
- 보급품 사용 페널티
- 조기 사망·중도 이탈 페널티
```

특별 목표 예시는 다음과 같다.

- 방향성 보호막을 교차 사격으로 2회 해제
- 문이 한 번도 파괴되지 않음
- 모든 플레이어 생존
- 제한된 포탑 수로 클리어
- 계약 대여 황금 포탑을 사용하지 않고 클리어

현재 알파 점수는 아래처럼 서버에서 계산한다. 한 계약은 재도전해도 최고 한 번의 점수만 시즌 합계에 반영한다.

```text
승리 7,500점 / 패배 1,200점
+ 시간 점수 (최대 1,200점)
+ 평균 문 HP 비율 점수 (최대 1,000점)
- 개인 보급품 사용 1개당 180점 (페널티 계약만)
```

시즌 순위는 시즌 점수 내림차순으로 정한다. 동점이면 아래 순서로 결정한다.

1. 더 높은 난이도 계약 클리어
2. 합산 남은 시간이 많음
3. 평균 문 잔여 HP가 높음
4. 사용한 보급품이 적음
5. 해당 점수에 먼저 도달

## 7. 기여도·사망·이탈

공격 피해만으로 기여도를 계산하지 않는다. 생존, 경제, 방어와 지원을 함께 기록한다.

- 생존한 전투 시간
- 건설과 업그레이드 투자
- 포탑 유효 피해
- 감속·속박으로 확보한 유효 시간
- 문 수리와 피해 감소
- 교차 사격과 특별 목표 기여
- 중도 이탈 여부

4인 랭크에서 정상 랭크 점수를 받으려면 전투 시간의 60% 이상 참여하고 팀 전체 기여도의 10% 이상을 달성해야 한다.

| 상황 | 혼자하기·친구랑하기 | 랭크전 |
| --- | --- | --- |
| 생존 상태로 승리 | 전체 일반 진행 보상 | 정상 계약 점수 |
| 후반 사망 후 팀 승리 | 전체 일반 진행 보상 | 참여·기여도에 따른 일부 점수 |
| 초반 사망 후 관전 | 전체 일반 진행 보상 | 계약 점수 0~20% |
| 중도 이탈 | 재접속 유예 후 패배 처리 | 점수 감소와 반복 이탈 제재 |
| 팀 패배 | 기존 패배 보상 | 기본 참여 점수만 지급 |

## 8. 랭크 티어와 순위표

일반 진행의 `하수 → 레전드`와 혼동되지 않는 별도 티어를 사용한다.

```text
브론즈 III → 브론즈 I
실버 III → 실버 I
골드 III → 골드 I
플래티넘 III → 플래티넘 I
다이아 III → 다이아 I
마스터
최강자 — 현재 시즌 상위 50명
```

순위표에는 다음 정보를 표시한다.

- 현재 순위, 닉네임, 대표 캐릭터
- 시즌 랭크 배지와 티어
- 시즌 점수와 최고 클리어 난이도
- 시즌 종료까지 남은 시간
- 상위 50명
- 내 순위 전후 5명
- 계약별 최고 점수와 사용한 보급품 수

## 9. 시즌 보상

아래 포인트는 S1 초기 제안값이며 시즌 경제 로그에 따라 다음 시즌부터 조정할 수 있다. 한 계정은 자신이 속한 가장 높은 보상 구간 하나만 받는다.

| 최종 순위 | 포인트 | 전투 보급품 | 한정 보상 |
| --- | ---: | ---: | --- |
| 1위 | 15,000P | 각 종류 10개 | 1위 전용 신규 스킨, 금색 왕관, `S1 시즌 최강자` 칭호 |
| 2~5위 | 8,000P | 각 종류 5개 | 상위 한정 신규 스킨, 은색 왕관, `S1 시즌 무법자` 칭호 |
| 6~20위 | 4,000P | 각 종류 3개 | 하위 한정 신규 스킨, 청동 왕관, `S1 시즌 추적자` 칭호 |
| 21~50위 | 2,500P | 각 종류 1개 | `S1 상위 50인` 프로필 테두리 |
| 상위 10% | 1,000P | 없음 | 시즌 상위권 배지 |
| 상위 25% | 600P | 없음 | 시즌 도전자 배지 |
| 배치 완료 | 300P | 없음 | 시즌 참가 기록 |

- 고정 순위 보상은 배치 완료 후 유효 계약을 10회 이상 플레이한 계정만 받을 수 있다.
- 참가자가 적은 초기 시즌에는 21~50위 보상에 `상위 25% 이내` 조건을 함께 적용해 사실상 전원 보상이 되지 않게 한다.
- 한정 스킨의 고유 능력은 혼자하기·친구랑하기에서 사용할 수 있지만 랭크전에서는 표준 능력치로 보정한다.
- 시즌 칭호와 왕관 라벨은 영구 보존하며 보유한 칭호 중 하나만 장착한다.
- 현재 시즌 1위에게는 시즌 종료 전까지 별도의 `현 시즌 최강자` 임시 효과를 표시할 수 있다.

### 9.1 칭호 라벨

- 1위: 금색 왕관, 금색·붉은색 애니메이션 그라데이션
- 2~5위: 은색 왕관, 은색·보라색 그라데이션
- 6~20위: 청동 왕관, 청동·주황색 그라데이션
- 홈, 친구 목록, 대기실과 인게임 이름표에 장착한 칭호를 표시한다.

### 9.2 시즌 아트 자산

- 시즌 티어 배지는 `public/assets/ranks/season-{bronze|silver|gold|platinum|diamond|master|challenger}.png`를 사용한다.
- 최종 보상 칭호 라벨에는 `crown-gold.png`, `crown-silver.png`, `crown-bronze.png`를 각각 사용한다.
- 홈 선택 상태와 시즌 랭킹 모달은 같은 배지·왕관 파일을 직접 참조한다. 점수 기반 티어가 바뀌어도 문자 아이콘으로 대체하지 않는다.

## 10. 시즌 종료와 보상 지급

- 시즌 종료 전에 시작한 경기는 종료 시각 이후 최대 10분 안에 끝나야 기록한다.
- 종료 뒤 순위표를 동결하고 중복 결과, 비정상 점수, 이탈 악용과 계정 공유 의심 기록을 검증한다.
- 검증 완료 후 서버가 최종 순위를 확정하고 보상을 계정 우편함으로 자동 지급한다.
- 우편 보상 수령 기간은 30일로 한다.
- 운영 제재로 순위가 무효화되면 아래 순위가 승계한다.
- 시즌 종료 시점의 순위표와 수상자 외형을 명예의 전당에 보존한다.

```ts
interface RankedSeason {
```

## 12. 구현 순서

1. 사용자 표시 명칭을 `혼자하기`, `친구랑하기`로 변경
2. 선택한 플레이 방식에 따른 홈·대기실·인게임 배지 표시
3. 일반 진행 보상과 랭크 시즌 저장소 분리
4. 14일 시즌과 48시간 계약 스케줄러
5. 고정 시드 계약과 황금 포탑 정책
6. 계약별 4인 대기열, 서버 점수 계산과 배치전·개인 리더보드
7. 시즌 동결·검증·보상 우편함·명예의 전당
8. 운영 데이터 검증 후 2인 고정 팀 랭크

## 13. 검증 조건

- 선택한 플레이 방식과 홈 프로필의 등급·배지가 일치한다.
- 같은 계약 ID는 모든 참가자에게 동일한 맵, 귀신, 규칙, 타임어택과 장비 정책을 생성한다.
- 개인 황금 티켓·영구 설계도·스킨 능력이 랭크 성능을 높이지 않는다.
- 재접속해도 계약 시드와 최고 점수가 바뀌지 않는다.
- 7개 계약 중 상위 5개만 시즌 점수에 반영된다.
- 일반 진행 XP·포인트와 랭크 시즌 점수는 직접 변환되지 않는다.
- 초반 사망 후 팀 승리한 플레이어가 생존자와 같은 랭크 점수를 받지 않는다.
- 수리·지원 역할도 기여도 조건을 충족할 수 있다.
- 시즌 종료 후 순위표가 동결되고 한 계정에 보상이 한 번만 지급된다.
- 시즌 한정 스킨의 고유 능력이 랭크전에서 표준화된다.

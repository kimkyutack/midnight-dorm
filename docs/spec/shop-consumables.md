## 1. 목적과 분리 원칙

전술 보급은 계정 포인트로 수량을 구매해 필요할 때 한 번 사용하는 소모품이다. 램프는 한 판 안에서 확률로 획득하는 지속형 보상이고, 전술 보급은 원하는 타이밍에 선택하는 위기 대응 도구다.

1. 전술 보급은 `RANDOM_ITEMS`의 ID·이름·영구 경제/전투 효과를 재사용하지 않는다.
2. 램프의 초당 골드·전력 생산, 영구 포탑 피해/공격속도/사거리 보정, 황금 티켓은 상점에서 판매하지 않는다.
3. 모든 보급품은 즉시 효과, 짧은 지속 효과 또는 **다음 한 번의 설비 업그레이드** 보조만 제공한다.
4. 외형 상품은 `상점 > 외형 상점`, 수량형 보급품은 `상점 > 전술 보급`에서 구매한다.

## 2. 구매·재고·사용 규칙

```text
포인트로 1개 또는 5개 구매
  → 구매 확인 모달에서 확정
  → 계정 전술 재고 +수량
  → 대기실에서 최대 3종 장착
  → 인게임 가방에서 성공적으로 사용
  → 해당 계정 재고 -1
```

- 같은 보급품은 구매 횟수 제한 없이 누적할 수 있다. 10개를 사면 성공 사용 10회까지 쓸 수 있다.
- 한 판에서 같은 보급품은 한 번만 사용한다. 실패한 대상 검증, 취소, 미사용 경기 종료에는 재고를 차감하지 않는다.
- 멀티에서 공동 문·공동 설비에 사용해도 재고와 구매 기록은 사용한 플레이어에게만 귀속한다.
- 서버가 가격, 보유 수량, 사용 가능 상태, 대상 유효성, 중복 요청을 판정한다. 클라이언트의 버튼은 표시와 요청만 담당한다.

## 4. UI·아트 규칙

- 외형 상점과 전술 보급의 구매 버튼은 모두 즉시 차감하지 않는다. 이름·수량·총 포인트를 보여 주는 확인 모달에서 `구매하기`를 눌러야 요청한다.
- 모든 카드에는 실제 게임 모델에서 파생한 3D 썸네일을 쓴다. 모바일 3열 카드에서 머리·귀·꼬리·도구가 잘리지 않도록 별도 축소 카메라 구도를 사용한다.
- 전술 보급은 3열 스크롤 그리드에 30종을 표시한다. 각 카드는 분류, 보유 수량, 1개 가격, 5개 구매를 제공한다.
- 보급품은 아이콘 문자만으로 식별하지 않는다. 렌즈·나침반·사탕·망토·호루라기·퍼티·리벳·봉인서·시계·공구 등 각 ID에 다른 3D 정물 모델을 제공한다.

## 5. 검증 기준

- `SHOP_CONSUMABLES`는 정확히 30개이며, ID가 중복되지 않고 램프 아이템 ID와 겹치지 않는다.
- 포인트가 부족하거나 구매 확인을 취소하면 재고와 포인트가 바뀌지 않는다.
- 같은 구매/사용 요청이 재전송돼도 서버에서 재고 차감과 효과가 한 번만 적용된다.
- 할인 보급은 선택한 설비의 다음 골드 업그레이드 한 번에만 적용되고, 적용 뒤 할인 대상과 비율을 모두 초기화한다.
- 320px 안팎의 세로 화면에서도 카드 이미지와 가격/구매 버튼이 카드 경계를 넘거나 잘리지 않는다.
